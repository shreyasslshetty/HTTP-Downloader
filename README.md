# HTTP-Downloader
This project is developed using Java and downloads any type files(.doc, .jpg, .png, etc.) from the internet in multiple parts and merge them when received within the system.

#EXECUTION METHOD:

1. Use your choice of CMD/Terminal
2. Navigate to SHETTYSHREYAS-http_downloader
3. Run JAR file: http_downloader.jar
4. Run Command: java -jar http_downloader.jar <URL> <Number of TCP Connections>
  - URL: The link of the file to be downloaded from the internet.
  - TCP Connections: Number parts/connections required to download file.
  
5. Other way to Run jar file: java -cp http_downloader.jar http_download.http_downloader <URL> <Number of TCP Connections>
